## ğŸ§© Gameplay Mechanics & Objectives
- The game provides **controls and objectives in-game**, so players learn as they progress.
- Your core objective is to:
  1. Explore the dungeon  
  2. **Solve riddles**  
  3. Use the **keywords** you discover to unlock chests  
- Keywords:
  - Are **alphanumeric**
  - **Do not appear on-screen** when typed (hidden input)
- There is **no time limit**, but:
  - **The first team to finish the game earns an advantage** in the event.
